# combat-ubuesque

Introduction :


It's lab project as game. It has for aim to make interaction between Angular 4 and Symfony 3. 

The client and display is manage by Angular while the data and login is manage by a Symfony (via a REST API).


Angular is in root directory.
Symfony is in backend directory.


Plug-in use :

Angular 4 package use : 
- Less

Symfony 3 bundle use : 
- FOS User
- FOS Rest
- JMS Serializer
- Lexik JWT Authentication
- Nelmio Cors


Information :

The game is in French.
The front is still under construction.
